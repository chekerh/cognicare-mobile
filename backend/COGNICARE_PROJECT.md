# CogniCare - Complete Cognitive Health Platform
A comprehensive Flutter + NestJS application with JWT authentication, MongoDB, and modern development practices.

## Quick Start
```bash
git clone <your-repo-url>
cd cognicare
docker-compose up -d
```

## Services
- Frontend: http://localhost:8080
- Backend API: http://localhost:3000
- API Docs: http://localhost:3000/api

Built with ❤️ using Flutter & NestJS
