# CogniCare Frontend

A Flutter application for personalized cognitive health and development.

## Features

- **Multi-language Support**: English, French, and Arabic (RTL)
- **Authentication**: JWT-based login/signup with secure storage
- **Onboarding Flow**: Interactive introduction for new users
- **Modern UI**: Clean design with accessibility features
- **Cross-platform**: iOS, Android, Web, Desktop support

## Getting Started

1. Ensure Flutter is installed: `flutter doctor`
2. Install dependencies: `flutter pub get`
3. Run the app: `flutter run`

## Backend

This frontend connects to a NestJS backend with MongoDB. Make sure the backend is running on `http://localhost:3000`.
